Lab 14 - Renderer and Refactoring

Overview

Over the last five labs you built six abstraction classes. This final lab in the series has two goals. First, you complete the Renderer class by adding the Draw() methods that were deferred in Lab 09 (GlCall Error System) because they reference VertexArray, IndexBuffer, and Shader - classes that did not exist at that point. Second, you refactor the Lab 08 monolithic main.cpp by replacing every raw OpenGL block with instantiations of those six classes. When the refactor is complete, main.cpp shrinks from over 100 lines of GL boilerplate to a concise, readable scene setup, and RAII eliminates the entire manual cleanup section at the end.

Learning Objectives

By the end of this lab you will be able to:

  • Explain why Renderer::Draw was deferred to this lab and which header includes it now requires
  • Implement Renderer::Draw by binding the shader, VAO, and IBO in the correct order before calling glDrawElements
  • Replace every raw glGenBuffers, glBufferData, and glVertexAttribPointer call in main.cpp with the corresponding class instances
  • Explain why the manual glDeleteBuffers, glDeleteTextures, and glDeleteProgram calls in the Lab 08 cleanup section can be removed entirely
  • Identify the one line that links the sampler2D uniform to a texture unit and explain why it only needs to be called once outside the render loop

Concept

The Renderer class was introduced in Lab 09 (GlCall Error System) with only its Clear() methods. Adding Draw() was deferred because its signature references VertexArray, IndexBuffer, and Shader - and Renderer.h cannot include headers that do not exist yet. Now that all three classes are built, Renderer.h can include them at the top and declare the Draw overloads. The implementation in Renderer.cpp binds the three objects in the required order - shader first, then VAO, then IBO - and calls glDrawElements with the index count from ib.GetCount(). Centralising this sequence in one method means that every test scene in the project can call renderer.Draw(vao, ibo, shader) without repeating the bind sequence.

The refactoring task demonstrates the value of RAII concretely. The Lab 08 main.cpp had a multi-line cleanup block at the end: glDeleteTextures, glDeleteBuffers (twice), glDeleteVertexArrays, glDeleteProgram. In the refactored version, all of that code disappears. When the local variables vao, vbo, ibo, shader, and tex go out of scope at the end of main, their destructors call the corresponding glDelete* functions automatically, in reverse declaration order. There is nothing to forget, nothing to leak, and nothing to call manually.

One subtle point in the refactored main.cpp is that shader.setUniform1i("u_Texture", 0) is called once before the render loop, not inside it. This call sets the GLSL sampler2D uniform to texture unit 0. Because the sampler unit does not change between frames, repeating the call every frame is redundant. The texture itself changes frame to frame only if you are animating it - here it does not, so tex.Bind(0) inside the loop is all that is needed. The rule is: set uniform values only as often as they actually change.

Task 2 - Implement Renderer::Draw

Open src/Renderer.cpp and add the two Draw implementations.

  1. Add the first overload. Bind the shader, then the VAO, then the IBO. Call glDrawElements with GL_TRIANGLES, ib.GetCount(), GL_UNSIGNED_INT, and nullptr (the IBO is already bound, so no offset pointer is needed): 
    void Renderer::Draw(const VertexArray& va, const IndexBuffer& ib, const Shader& shader) const
{
    shader.Bind();
    va.Bind();
    ib.Bind();
    GlCall(glDrawElements(GL_TRIANGLES, ib.GetCount(), GL_UNSIGNED_INT, nullptr));
}
  2. Add the second overload (no shader parameter - bind VAO and IBO only): 
    void Renderer::Draw(const VertexArray& va, const IndexBuffer& ib) const
{
    va.Bind();
    ib.Bind();
    GlCall(glDrawElements(GL_TRIANGLES, ib.GetCount(), GL_UNSIGNED_INT, nullptr));
}
  3. Build the project. There should be no new errors at this point.
src/Renderer.cpp
#include "Renderer.h"
#include <iostream>
#include <algorithm>

void glClearError()
{
    while (glGetError() != GL_NO_ERROR);
}

bool glLogCall(const char* function, const char* file, int line)
{
    while (GLenum error = glGetError())
    {
        std::cout << "[OpenGL Error]( " << error << " ) "
                  << function << " File: " << file << " Line:" << line << std::endl;
        return false;
    }
    return true;
}

void Renderer::Draw(const VertexArray& va, const IndexBuffer& ib, const Shader& shader) const
{
    shader.Bind();
    va.Bind();
    ib.Bind();
    GlCall(glDrawElements(GL_TRIANGLES, ib.GetCount(), GL_UNSIGNED_INT, nullptr));
}

void Renderer::Draw(const VertexArray& va, const IndexBuffer& ib) const
{
    va.Bind();
    ib.Bind();
    GlCall(glDrawElements(GL_TRIANGLES, ib.GetCount(), GL_UNSIGNED_INT, nullptr));
}

void Renderer::Clear() const
{
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
}

void Renderer::Clear(std::string colour) const
{
    std::transform(colour.begin(), colour.end(), colour.begin(), ::towlower);
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);
    if (colour == "black")   GlCall(glClearColor(0, 0, 0, 1));
    if (colour == "white")   GlCall(glClearColor(1, 1, 1, 1));
    if (colour == "red")     GlCall(glClearColor(1, 0, 0, 1));
    if (colour == "green")   GlCall(glClearColor(0, 1, 0, 1));
    if (colour == "blue")    GlCall(glClearColor(0, 0, 1, 1));
    if (colour == "yellow")  GlCall(glClearColor(1, 1, 0, 1));
    if (colour == "cyan")    GlCall(glClearColor(0, 1, 1, 1));
    if (colour == "magenta") GlCall(glClearColor(1, 0, 1, 1));
}

void Renderer::ClearColour_Black() const
{
    GlCall(glClearColor(0, 0, 0, 1));
}

void Renderer::ClearColour_White() const
{
    GlCall(glClearColor(1, 1, 1, 1));
}

struct ShaderSource { std::string vertex; std::string fragment; };

ShaderSource parseShader(const std::string& filepath) {
    std::ifstream file(filepath);
    std::stringstream ss[2];
    int type = -1;
    std::string line;
    while (std::getline(file, line)) {
        if      (line.find("#shader vertex")   != std::string::npos) type = 0;
        else if (line.find("#shader fragment") != std::string::npos) type = 1;
        else if (type == 0 || type == 1) ss[type] << line << '\n';
    }
    return { ss[0].str(), ss[1].str() };
}

static unsigned int compileShader(unsigned int type, const std::string& source) {
    unsigned int id = glCreateShader(type);
    const char* src = source.c_str();
    glShaderSource(id, 1, &src, nullptr);
    glCompileShader(id);
    int result;
    glGetShaderiv(id, GL_COMPILE_STATUS, &result);
    if (!result) {
        int len; glGetShaderiv(id, GL_INFO_LOG_LENGTH, &len);
        char* msg = new char[len];
        glGetShaderInfoLog(id, len, &len, msg);
        std::cerr << "Shader compile error: " << msg << "\n";
        delete[] msg; glDeleteShader(id); return 0;
    }
    return id;
}

unsigned int createProgram(const std::string& vertSrc, const std::string& fragSrc) {
    unsigned int program = glCreateProgram();
    unsigned int vs = compileShader(GL_VERTEX_SHADER,   vertSrc);
    unsigned int fs = compileShader(GL_FRAGMENT_SHADER, fragSrc);
    glAttachShader(program, vs); glAttachShader(program, fs);
    glLinkProgram(program);
    glDeleteShader(vs); glDeleteShader(fs);
    return program;
}
