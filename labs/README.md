## Moodle upload instructions

1. In your Moodle course, add a File resource for each lab.
2. Upload the HTML file. Set Display to "In frame" or "Open" (not Force download).
3. Upload the matching solution HTML as a separate File resource labelled
   "[Lab title] - Solution". Make it available but encourage students to attempt
   the lab before consulting it.
